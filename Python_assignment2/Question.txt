Task
The provided code stub read and integer n, from STDIN , for all non negative integers i < n print i ^2 

Example
 n=3  
 
 the list of non-negative integers that are less than n=3 [0, 1, 2] print the square of each number on a seperate line.  
 
 0
 1
 4
 
Input Format:

 the first and only line contains the integer n  
 
Constraints:

1<n<20

Output Format:

 print n lines , one corresponding to each i   
 
Sample input 0:

 5
 
Sample Ouput 0:
 
 0
 1
 4
 9
 16
